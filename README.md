## Overview
This project is a Python-based simulation tool developed to visualize fundamental digital modulation techniques used in telecommunications. It generates and plots the time-domain waveforms for **Amplitude Shift Keying (ASK)**, **Frequency Shift Keying (FSK)**, and **Phase Shift Keying (PSK)**.

## Features
- **Binary Data Generation:** Simulates random bitstreams (Logic 0/1).
- **Carrier Wave Synthesis:** Generates high-frequency sinusoidal carrier signals using `NumPy`.
- **Modulation Algorithms:** Implements mathematical models to modulate the carrier based on binary input.
- **Visualization:** Uses `Matplotlib` to render clear comparisons between the input data, carrier, and modulated signals.

## Tech Stack
- **Language:** Python 3.x
- **Libraries:** `numpy`, `matplotlib`

## How to Run
1. Clone the repository.
2. Run `python modulation_visualizer.py`.
